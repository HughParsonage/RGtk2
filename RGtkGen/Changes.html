Version 2.0-0

Package pretty much rewritten for GObject-based libs (ie GTK2). Supports new defs file format.

Version 0.3-1

Put the trailing NULL entry in the registration table for the C routines.

Version 0.3-0

Register the routines in the Gtk library that have no interface but we can call directly. This is needed on systems for which the dynamic loader does not provide access to secondary libraries.

Version 0.2-0

  • Add show and .flush to the collapsed constructor functions that we generate if we are dealing with a GtkWidget or decsendant.
  • Added an extra new line at the end of the -internal.Rd
  • Enumerate the entire collection of R objects that are automatically generated in the return from generateCodeFiles() and these can be put into -internal.Rd files.

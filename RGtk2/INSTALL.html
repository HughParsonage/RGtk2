The RGtk2 package works with gtk-2.0.

For Windows, see the README.windows and INSTALL.windows files.

The RGtk package supports the Gtk and Gdk libraries, along with ATK, Pango, and (to a lesser extent) GObject/GLib. It optionally supports GtkMozEmbed and libglade. The configuration for the RGtk package looks for pkg-config to compute the compiler and linker flags needed to create the shared libraries.
